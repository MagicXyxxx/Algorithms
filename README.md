# Algorithms	算法源码合辑

## bitMapSort <br>
* 使用bit进行排序<br>
* 基于位图排序，实现了一个输入任意不重复数据的程序<br>


## one-dim pattern identification(连续向量最大和) <br>
* 使用简单扫描算算法、迭代、分治、动态规划算法实现了该功能<br>
* 对项目算法流程的优化改进与比较<br>


## Min_recursion <br>
* 使用递归实现的数组最小值的分治解法<br>
* 对代码部分功能的封装<br>

## CoinChange <br>
* 使用动态规划解决的换零钱问题<br>
* 对C语言可变参数的使用<br>
